# Corporations

Bachelor of Science HES-SO - Semester project 2013-14 @ [HE-Arc](http://www.he-arc.ch/)

Corporations is a mobile game for both Android and iOS. The main goal is to force the player to travel in real life to win. The goal of the player is to capture as many territories as possible to conquer the world.
